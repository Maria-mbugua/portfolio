# Portfolio

This project is a personal portfolio website built with modern web technologies.

## Live Demo

You can view the live application here: [https://portfolio-teal-sigma-fxwwkma4c1.vercel.app/](https://portfolio-teal-sigma-fxwwkma4c1.vercel.app/)

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**
   ```sh
   git clone https://github.com/Maria-mbugua/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```

4. **Build for production**
   ```sh
   npm run build
   ```
